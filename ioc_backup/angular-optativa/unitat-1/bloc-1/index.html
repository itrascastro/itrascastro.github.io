Introducció a Angular

Angular és un framework de desenvolupament per a aplicacions web escrit en TypeScript. Desenvolupat per Google, Angular permet crear aplicacions d'una sola pàgina (SPA) robustes i escalables.

Informació
Definició: Angular és un framework complet que proporciona una estructura i eines per desenvolupar aplicacions completes, mentre que una biblioteca com React només proporciona funcions específiques.

Característiques Principals

  • TypeScript: Llenguatge principal amb tipatge fort
  • Components: Arquitectura basada en components reutilitzables
  • CLI: Eina de línia de comandaments potent
  • Ecosystem complet: Routing, HTTP client, forms integrats

Comparació de Frameworks

Framework Tipus Llenguatge Arquitectura
Angular Framework complet TypeScript MVC + Components
React Llibreria UI JavaScript/JSX Component-based
Vue.js Framework progressiu JavaScript MVVM

Configuració de l'Entorn

Abans de començar, necessitem configurar el nostre entorn de desenvolupament.

Important
Prerequisit: Necessites tenir Node.js v18+ i npm v9+ instal·lats al sistema.

Verificació del Sistema

BASH 

# Verificació completa del sistema de desenvolupament Angular
echo '=== VERIFICACIÓ DEL SISTEMA ANGULAR ==='
echo 'Comprovant versions dels components necessaris...'

# Node.js - requereix versió 18 o superior per Angular 17+
echo -n 'Node.js: ' && node --version || echo 'ERROR: Node.js no instal·lat'

# npm - gestor de paquets per defecte
echo -n 'npm: ' && npm --version || echo 'ERROR: npm no disponible'

# Verificar si Angular CLI està instal·lat globalment
if command -v ng &> /dev/null; then
    echo 'Angular CLI: Instal·lat correctament'
    ng version --skip-git 2>/dev/null | grep -E '(Angular CLI|Package)' | head -3
else
    echo 'ADVERTÈNCIA: Angular CLI no instal·lat globalment'
    echo 'Executar: npm install -g @angular/cli@latest'
fi

# Verificació opcional de TypeScript global (recomanat per desenvolupament)
if command -v tsc &> /dev/null; then
    echo -n 'TypeScript global: ' && tsc --version
else
    echo 'INFO: TypeScript global no instal·lat (opcional)'
fi

echo '=== VERIFICACIÓ COMPLETADA ==='

Instal·lació d'Angular CLI

BASH 

# Instal·lació d'Angular CLI amb configuració completa
npm install -g @angular/cli@latest

# Verificar instal·lació i mostrar versions de tot l'ecosistema
ng version

# Configuració opcional del CLI per millorar experiència de desenvolupament
ng config -g cli.analytics false  # Desactivar telemetria per privacitat
ng config -g cli.warnings.versionMismatch false  # Evitar alertes de versions menors

echo 'Angular CLI instal·lat i configurat correctament!'
Èxit
CLI instal·lat: Si la comanda ng version mostra informació d'Angular, l'eina està llesta per usar.

Verificació de la Configuració

Passos de Verificació

La comanda ng version mostra informació
L'entorn està llest per desenvolupar

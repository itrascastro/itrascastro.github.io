Aquest bloc avança en l'estudi de les estructures de dades introduint les col·leccions del framework de Java. Explorarem les principals interfícies i implementacions que permeten gestionar dades de manera més flexible i eficient que els arrays tradicionals.

Les col·leccions de Java proporcionen estructures dinàmiques que poden créixer o decreixe segons les necessitats del programa, oferint operacions avançades de cerca, inserció, eliminació i ordenació.

Els estudiants aprendran a utilitzar List, Set, Map i les seves implementacions més importants com ArrayList, LinkedList, HashSet, TreeSet, HashMap i TreeMap, així com els algorismes associats per treballar amb aquestes estructures de manera eficient.

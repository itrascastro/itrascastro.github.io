Unitat 6

Aplicació de les estructures d'emmagatzematge (RA6)

Descripció de la Unitat

Objectius: Aquesta unitat cobreix l'aplicació de les estructures d'emmagatzematge en Java, des dels arrays bàsics fins a les col·leccions avançades del framework, essencials per organitzar i gestionar dades eficientment.

  • Dominar arrays unidimensionals i bidimensionals amb operacions de manipulació
  • Implementar algorismes fonamentals de cerca i ordenació
  • Utilitzar col·leccions del framework Java: List, Set i Map
  • Aplicar criteris de rendiment per seleccionar l'estructura òptima
  • Treballar amb generics per garantir seguretat de tipus
  • Processar dades amb streams i programació funcional

Resultat d'Aprenentatge: RA6 - Aplicació de les estructures d'emmagatzematge, fonamental per a la gestió eficient de dades i el disseny d'algorismes.

6.1

Estructures de dades bàsiques

Què aprendràs:

Aquest bloc introdueix les estructures de dades bàsiques en Java, centrant-se en l'ús d'arrays unidimensionals i bidimensionals. Aprendràs a crear, gestionar i manipular arrays per organitzar dades de manera eficient.

Els continguts inclouen declaració i inicialització d'arrays, operacions bàsiques com recorregut i cerca, i la implementació d'algorismes fonamentals per treballar amb aquestes estructures de dades.

6.2

Col·leccions i tipus de dades avançats

Què aprendràs:

Aquest bloc avança en l'estudi de les estructures de dades introduint les col·leccions del framework de Java. Explorarem les principals interfícies i implementacions que permeten gestionar dades de manera més flexible i eficient que els arrays tradicionals.

Les col·leccions de Java proporcionen estructures dinàmiques que poden créixer o decreixe segons les necessitats del programa, oferint operacions avançades de cerca, inserció, eliminació i ordenació.

Els estudiants aprendran a utilitzar List, Set, Map i les seves implementacions més importants com ArrayList, LinkedList, HashSet, TreeSet, HashMap i TreeMap, així com els algorismes associats per treballar amb aquestes estructures de manera eficient.

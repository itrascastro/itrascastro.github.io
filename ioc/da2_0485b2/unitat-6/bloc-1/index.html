Introducció a les Estructures de Dades

Les estructures de dades són fonamentals en programació per organitzar, gestionar i emmagatzemar informació de manera eficient. En aquest bloc introductori explorarem els arrays, que són la base de moltes altres estructures de dades més complexes.

Què són les Estructures de Dades?

Una estructura de dades és una forma particular d'organitzar i emmagatzemar dades per facilitar l'accés i les modificacions posteriors. Les estructures de dades defineixen:

  • Com s'organitzen les dades: La disposició física o lògica dels elements
  • Com s'accedeix a les dades: Els mètodes per llegir o modificar elements
  • Quines operacions es poden realitzar: Inserció, eliminació, cerca, ordenació, etc.
  • L'eficiència temporal i espacial: Cost computacional de les operacions

Classificació de les Estructures de Dades

Les estructures de dades es poden classificar en diferents categories:

Tipus Característiques Exemples
Primitives Tipus bàsics del llenguatge int, double, boolean, char
Lineals Elements organitzats seqüencialment Arrays, Llistes, Piles, Cues
No Lineals Relacions jeràrquiques o de xarxa Arbres, Grafs, Taules Hash
Estàtiques Mida fixa en temps de compilació Arrays tradicionals
Dinàmiques Mida variable en temps d'execució ArrayList, LinkedList, HashMap

Els Arrays: Estructures Lineals Estàtiques

Els arrays són la forma més bàsica d'estructura de dades lineal. Consisteixen en una col·lecció d'elements del mateix tipus, emmagatzemats en posicions de memòria consecutives i accessibles mitjançant un índex numèric.

Definició

Array: Estructura de dades que conté una col·lecció d'elements del mateix tipus, organitzats seqüencialment i accessibles mitjançant un índex numèric que comença en 0.

Avantatges i Inconvenients dels Arrays

Avantatges:

  • Accés directe: Temps constant O(1) per accedir a qualsevol element
  • Eficiència de memòria: Emmagatzematge contigu sense sobrecàrrega
  • Simplicitat: Fàcils d'entendre i implementar
  • Compatibilitat: Suport natiu en gairebé tots els llenguatges

Inconvenients:

  • Mida fixa: No es pot canviar la mida després de la creació
  • Inserció/eliminació costosa: Requereix desplaçar elements O(n)
  • Memòria preasignada: Pot malbaratar espai si no s'usa completament
  • Tipus homogeni: Tots els elements han de ser del mateix tipus
Informació
Objectius d'aquest bloc:

• Dominar les operacions bàsiques: creació, accés, recorregut

Memòria i Rendiment dels Arrays

Els arrays són extremadament eficients gràcies a la seva organització en memòria:

  • Localitat espacial: Elements consecutius ocupen posicions adjacents de memòria
  • Accés indexat: Fórmula matemàtica: direcció = base + (índex × mida_element)
  • Cache-friendly: Aprofiten la memòria cau del processador per millor rendiment
  • Predicibilitat: El compilador pot optimitzar l'accés als arrays

Arrays Unidimensionals

Declaració i Inicialització

Un array es pot declarar de diferents maneres:

JAVA 

int[] numeros = new int[5]; // Array sense inicialitzar
int[] valors = {1, 5, 10, -3, 8}; // Array inicialitzat

Accés als Elements

Els elements d'un array s'accedeixen mitjançant el seu índex:

JAVA 

System.out.println("Primer element: " + valors[0]); // Mostra 1
valors[2] = 100; // Modifica el valor a la posició 2
Definició

Índex: Posició d'un element dins de l'array. Els índexs comencen sempre en 0 i van fins a la longitud de l'array menys 1.

Recorregut d'un Array

Per recórrer tots els elements d'un array utilitzem bucles:

JAVA 

public static void imprimirArray(int[] a) {
    for (int i = 0; i < a.length; i++) {
        System.out.print(a[i] + " ");
    }
    System.out.println();
}

Cerca en un Array

Implementació d'un algorisme de cerca lineal:

JAVA 

public static void buscar(int[] a, int n) {
    boolean trobat = false;
    int i = 0;
    while (!trobat && i < a.length) {
        if (a[i] == n) {
            trobat = true;
        }
        i++;
    }
    if (trobat) {
        System.out.println("Element trobat a la posició " + (i - 1));
    } else {
        System.out.println("Element no trobat");
    }
}

Suma dels Elements d'un Array

JAVA 

public static void sumaArray(int[] a) {
    int suma = 0;
    for (int i = 0; i < a.length; i++) {
        suma += a[i];
    }
    System.out.println("Suma total: " + suma);
}

Rellenar un Array amb Valors Aleatoris

JAVA 

import java.util.Random;

public static void rellenarAleatorios(int[] a) {
    Random aleatori = new Random();
    for (int i = 0; i < a.length; i++) {
        a[i] = aleatori.nextInt(100); // Nombre entre 0 i 99
    }
}

Arrays Bidimensionals

Els arrays bidimensionals (matrius) permeten organitzar dades en files i columnes:

Declaració i Inicialització

JAVA 

int[][] matriu = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Accés als Elements

JAVA 

System.out.println(matriu[1][2]); // Mostra 6
matriu[0][1] = 20; // Modifica el valor a la posició [0][1]
Definició

Matriu: Array bidimensional que organitza elements en files i columnes. S'accedeix als elements amb dos índexs: [fila][columna].

Imprimir una Matriu

JAVA 

public static void imprimirMatriu(int[][] a) {
    for (int i = 0; i < a.length; i++) {
        for (int j = 0; j < a[i].length; j++) {
            System.out.print(a[i][j] + " ");
        }
        System.out.println();
    }
}

Modificació de la Diagonal Principal

JAVA 

public static void modificarDiagonal(int[][] a) {
    for (int i = 0; i < a.length; i++) {
        a[i][i] = 1;
    }
}

Suma de Tots els Elements

JAVA 

public static int sumaMatriu(int[][] a) {
    int suma = 0;
    for (int i = 0; i < a.length; i++) {
        for (int j = 0; j < a[i].length; j++) {
            suma += a[i][j];
        }
    }
    return suma;
}

Suma per Files i Columnes

JAVA 

public static void sumarFilesColumnes(int[][] a) {
    for (int i = 0; i < a.length; i++) {
        int sumaFil = 0;
        int sumaCol = 0;
        for (int j = 0; j < a[i].length; j++) {
            sumaFil += a[i][j];
            sumaCol += a[j][i];
        }
        System.out.println("Fila " + i + " suma: " + sumaFil);
        System.out.println("Columna " + i + " suma: " + sumaCol);
    }
}

Algorismes de Cerca i Ordenació

Els arrays són la base per implementar algorismes fonamentals de cerca i ordenació. Aquests algorismes són essencials per gestionar i processar dades de manera eficient.

Informació
Els algorismes de cerca i ordenació són fonamentals en programació. La cerca lineal té complexitat O(n), mentre que la cerca binària té O(log n) però requereix que l'array estigui ordenat.

Algorismes de Cerca

Cerca Lineal (Seqüencial)

La cerca lineal recorre l'array element per element fins trobar el valor buscat:

JAVA 

public static int cercaLineal(int[] array, int valor) {
    for (int i = 0; i < array.length; i++) {
        if (array[i] == valor) {
            return i; // Retorna l'índex on s'ha trobat
        }
    }
    return -1; // No trobat
}

// Versió que retorna informació detallada
public static void cercaLinealDetallada(int[] array, int valor) {
    int comparacions = 0;
    boolean trobat = false;
    int posicio = -1;
    
    for (int i = 0; i < array.length; i++) {
        comparacions++;
        if (array[i] == valor) {
            trobat = true;
            posicio = i;
            break;
        }
    }
    
    System.out.println("Valor " + valor + ": " + (trobat ? "trobat a posició " + posicio : "no trobat"));
    System.out.println("Comparacions realitzades: " + comparacions);
}
Definició

Cerca Lineal: Algorisme que busca un element recorrent l'array des del primer element fins al últim. Complexitat temporal: O(n).

Cerca Binària

La cerca binària funciona només amb arrays ordenats, dividint repetidament l'espai de cerca per la meitat:

JAVA 

public static int cercaBinaria(int[] array, int valor) {
    int esquerra = 0;
    int dreta = array.length - 1;
    
    while (esquerra <= dreta) {
        int mig = (esquerra + dreta) / 2;
        
        if (array[mig] == valor) {
            return mig; // Trobat!
        } else if (array[mig] < valor) {
            esquerra = mig + 1; // Buscar a la meitat dreta
        } else {
            dreta = mig - 1; // Buscar a la meitat esquerra
        }
    }
    
    return -1; // No trobat
}

// Versió recursiva
public static int cercaBinariaRecursiva(int[] array, int valor, int esquerra, int dreta) {
    if (esquerra > dreta) {
        return -1; // No trobat
    }
    
    int mig = (esquerra + dreta) / 2;
    
    if (array[mig] == valor) {
        return mig;
    } else if (array[mig] < valor) {
        return cercaBinariaRecursiva(array, valor, mig + 1, dreta);
    } else {
        return cercaBinariaRecursiva(array, valor, esquerra, mig - 1);
    }
}
Definició

Cerca Binària: Algorisme eficient per buscar en arrays ordenats que divideix l'espai de cerca per la meitat en cada iteració. Complexitat temporal: O(log n).

Algorismes d'Ordenació

Ordenació Bombolla (Bubble Sort)

Compara elements adjacents i els intercanvia si estan en l'ordre incorrecte:

JAVA 

public static void ordenaBombolla(int[] array) {
    int n = array.length;
    boolean intercanviat;
    
    for (int i = 0; i < n - 1; i++) {
        intercanviat = false;
        
        // Últims i elements ja estan ordenats
        for (int j = 0; j < n - 1 - i; j++) {
            if (array[j] > array[j + 1]) {
                // Intercanvi
                int temp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = temp;
                intercanviat = true;
            }
        }
        
        // Si no hi ha hagut intercanvis, l'array ja està ordenat
        if (!intercanviat) {
            break;
        }
    }
}

// Versió amb informació de debug
public static void ordenaBombollaDetallada(int[] array) {
    int n = array.length;
    int passades = 0;
    int intercanvis = 0;
    
    for (int i = 0; i < n - 1; i++) {
        passades++;
        boolean intercanviat = false;
        
        System.out.println("Passada " + passades + ": " + java.util.Arrays.toString(array));
        
        for (int j = 0; j < n - 1 - i; j++) {
            if (array[j] > array[j + 1]) {
                int temp = array[j];
                array[j] = array[j + 1];
                array[j + 1] = temp;
                intercanviat = true;
                intercanvis++;
            }
        }
        
        if (!intercanviat) break;
    }
    
    System.out.println("Resultat final: " + java.util.Arrays.toString(array));
    System.out.println("Passades: " + passades + ", Intercanvis: " + intercanvis);
}

Ordenació per Selecció (Selection Sort)

Troba repetidament l'element mínim i l'ubica al principi:

JAVA 

public static void ordenaSeleccio(int[] array) {
    int n = array.length;
    
    for (int i = 0; i < n - 1; i++) {
        // Trobar l'índex del mínim element
        int indexMin = i;
        for (int j = i + 1; j < n; j++) {
            if (array[j] < array[indexMin]) {
                indexMin = j;
            }
        }
        
        // Intercanviar l'element mínim amb el primer element
        if (indexMin != i) {
            int temp = array[i];
            array[i] = array[indexMin];
            array[indexMin] = temp;
        }
    }
}

Ordenació per Inserció (Insertion Sort)

Construeix l'array ordenat element per element, inserint cada nou element en la seva posició correcta:

JAVA 

public static void ordenaInsercio(int[] array) {
    for (int i = 1; i < array.length; i++) {
        int clau = array[i];
        int j = i - 1;
        
        // Moure elements majors que 'clau' una posició endavant
        while (j >= 0 && array[j] > clau) {
            array[j + 1] = array[j];
            j--;
        }
        
        array[j + 1] = clau;
    }
}

// Versió detallada que mostra el procés
public static void ordenaInsercioDetallada(int[] array) {
    System.out.println("Array inicial: " + java.util.Arrays.toString(array));
    
    for (int i = 1; i < array.length; i++) {
        int clau = array[i];
        int j = i - 1;
        
        System.out.println("Inserint " + clau + " en posició correcta...");
        
        while (j >= 0 && array[j] > clau) {
            array[j + 1] = array[j];
            j--;
        }
        
        array[j + 1] = clau;
        System.out.println("Després del pas " + i + ": " + java.util.Arrays.toString(array));
    }
}

Comparació d'Algorismes

Algorisme Complexitat Millor Cas Complexitat Cas Mitjà Complexitat Pitjor Cas Espai Estable
Cerca Lineal O(1) O(n) O(n) O(1) N/A
Cerca Binària O(1) O(log n) O(log n) O(1) N/A
Bombolla O(n) O(n²) O(n²) O(1) Sí
Selecció O(n²) O(n²) O(n²) O(1) No
Inserció O(n) O(n²) O(n²) O(1) Sí
Definició

Algorisme Estable: Un algorisme d'ordenació és estable si manté l'ordre relatiu d'elements amb claus iguals.

Exemple Pràctic Complet

JAVA 

public class AlgorismesArrays {
    public static void main(String[] args) {
        // Array de prova
        int[] numeros = {64, 34, 25, 12, 22, 11, 90, 5};
        
        System.out.println("=== PROVES DE CERCA ===");
        System.out.println("Array: " + java.util.Arrays.toString(numeros));
        
        // Cerca lineal
        int valorBuscat = 22;
        int resultat = cercaLineal(numeros, valorBuscat);
        System.out.println("Cerca lineal de " + valorBuscat + ": " + 
                          (resultat != -1 ? "trobat a posició " + resultat : "no trobat"));
        
        System.out.println("\n=== PROVES D'ORDENACIÓ ===");
        
        // Provar cada algorisme amb còpia de l'array
        int[] copia1 = numeros.clone();
        int[] copia2 = numeros.clone();
        int[] copia3 = numeros.clone();
        
        System.out.println("Array original: " + java.util.Arrays.toString(numeros));
        
        ordenaBombolla(copia1);
        System.out.println("Bombolla: " + java.util.Arrays.toString(copia1));
        
        ordenaSeleccio(copia2);
        System.out.println("Selecció: " + java.util.Arrays.toString(copia2));
        
        ordenaInsercio(copia3);
        System.out.println("Inserció: " + java.util.Arrays.toString(copia3));
        
        // Provar cerca binària amb array ordenat
        System.out.println("\n=== CERCA BINÀRIA ===");
        int resultatBinari = cercaBinaria(copia1, valorBuscat);
        System.out.println("Cerca binària de " + valorBuscat + " en array ordenat: " + 
                          (resultatBinari != -1 ? "trobat a posició " + resultatBinari : "no trobat"));
    }
}
Important
Important: Sempre verifica els límits de l'array per evitar ArrayIndexOutOfBoundsException. L'últim índex vàlid és sempre array.length - 1.

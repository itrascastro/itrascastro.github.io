Introducció

Contingut multimèdia d'aquesta lectura
Veure vídeo Escoltar àudio

En aquest cas d'ús pràctic, integrareu tots els conceptes treballats a la Unitat 2 per construir una aplicació completa de catàleg d'elements amb Angular. Desenvolupareu una interfície interactiva que inclou components personalitzats, gestió d'esdeveniments, formularis reactius i funcionalitats d'accessibilitat.

Aquest cas d'ús representa un projecte realista que us permetrà aplicar els coneixements adquirits sobre components, comunicació entre components, directives, pipes, formularis i accessibilitat en un context integrat.

Objectius

En finalitzar aquest cas d'ús, hauríeu de ser capaços de:

  • Crear components Angular reutilitzables amb encapsulació adequada
  • Implementar la comunicació entre components pare-fill utilitzant @Input i @Output
  • Desenvolupar formularis reactius amb validacions personalitzades
  • Aplicar directives estructurals i d'atributs per controlar la renderització
  • Gestionar diferents estats de la interfície (càrrega, error, buit, contingut)
  • Implementar característiques d'accessibilitat complint amb les pautes WCAG

Criteris d'avaluació (RA2)

Aquest cas d'ús cobreix els següents criteris del Resultat d'Aprenentatge 2:

  • RA2.1: S'han identificat i utilitzat components reutilitzables
  • RA2.2: S'han implementat mecanismes de comunicació entre components
  • RA2.3: S'han creat i validat formularis reactius
  • RA2.4: S'han aplicat directives per manipular el DOM de manera declarativa
  • RA2.5: S'han implementat característiques d'accessibilitat web

  1. Component de targeta — Mostra informació d'un element (títol, descripció, imatge, valoració)

Aspectes clau del codi:

  • grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)) — Graella responsive automàtica
  • Media query per mòbil amb una sola columna

import { Component, EventEmitter, Output } from '@angular/core';
import { CommonModule } from '@angular/common';
import { FormsModule } from '@angular/forms';

@Component({
  selector: 'app-barra-cerca',
  standalone: true,
  imports: [CommonModule, FormsModule],
  template: `
    <form class="formulari-cerca" (ngSubmit)="cercar()" #formCerca="ngForm">
      <div class="grup-camp">
        <label for="camp-cerca">Cerca elements:</label>
        <div class="contenidor-input">
          <input type="text" id="camp-cerca" name="cerca" [(ngModel)]="textCerca" #campCerca="ngModel" minlength="3" placeholder="Mínim 3 caràcters..." aria-describedby="ajuda-cerca error-cerca" class="input-cerca" [class.invalid]="campCerca.invalid && campCerca.touched">

          <button type="submit" [disabled]="formCerca.invalid" class="boto-cerca" aria-label="Cercar elements">
            <i class="bi bi-search" aria-hidden="true"></i> Cercar
          </button>

          <button *ngIf="textCerca" type="button" (click)="netejar()" class="boto-netejar" aria-label="Esborrar cerca">
            <i class="bi bi-x-lg" aria-hidden="true"></i>
          </button>
        </div>

          Escriu almenys 3 caràcters per cercar
        </small>

      background: #e0e0e0;
    }

      color: #666;
      font-size: 0.875rem;

      color: #d32f2f;
      font-size: 0.875rem;
      font-weight: 500;
    }

    @media (max-width: 768px) {
      .contenidor-input {
        flex-direction: column;
      }
      .boto-cerca {
        width: 100%;
      }
    }
  `]
})
export class BarraCercaComponent {
  @Output() cercaCanviada = new EventEmitter<string>();

  textCerca = '';

  cercar(): void {
    if (this.textCerca.length >= 3) {
      this.cercaCanviada.emit(this.textCerca);
    }
  }

  netejar(): void {
    this.textCerca = '';
    this.cercaCanviada.emit('');
  }
}

Aspectes clau del codi:

  • @Output() cercaCanviada — Emet esdeveniments cap al component pare
  • [(ngModel)]="textCerca" — Vinculació bidireccional del camp
  • minlength="3" — Validació mínima de 3 caràcters
  • [disabled]="formCerca.invalid" — Botó deshabilitat si formulari invàlid
  • role="alert" aria-live="assertive" — Accessibilitat per a missatges d'error
  • Icones Bootstrap bi-search i bi-x-lg
  • Botó netejar només visible quan hi ha text

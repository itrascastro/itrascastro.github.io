Introducció

{% include multimedia-youtube.html unitat=1 bloc=4 seccio=0 %}

En aquest cas d'ús pràctic, integrareu tots els conceptes treballats a la Unitat 1 per preparar l'entorn complet de desenvolupament Angular. Configurareu les eines necessàries, creareu el vostre primer projecte i el publicareu a GitHub, establint així les bases per a tot el curs.

Aquest cas d'ús representa un flux de treball realista que us permetrà aplicar els coneixements adquirits sobre instal·lació d'eines, configuració de l'entorn, generació de projectes i control de versions en un context integrat.

Objectius

En finalitzar aquest cas d'ús, hauríeu de ser capaços de:

  • Verificar i configurar l'entorn tècnic complet (Node.js, npm, Angular CLI, VS Code, Git)
  • Generar el projecte base cataleg-elements amb estructura funcional
  • Validar servidor de desenvolupament amb hot reload operatiu
  • Publicar el projecte a GitHub amb control de versions actiu
  • Documentar el procés i preparar el projecte per a les següents unitats

Criteris d'avaluació (RA1)

Aquest cas d'ús cobreix els següents criteris del Resultat d'Aprenentatge 1:

  • RA1.1: Identifica les eines imprescindibles i comprova que compleixen els requisits del sistema
  • RA1.2: Instal·la i configura l'entorn de desenvolupament amb les eines i dependències necessàries
  • RA1.3: Crea l'estructura base del projecte organitzant carpetes i fitxers de manera coherent
  • RA1.4: Configura els fitxers d'execució i comprova que l'entorn arrenca correctament
  • RA1.5: Verifica el funcionament global amb proves o exemples senzills

4.7. Troubleshooting

Problema Causa Solució
git: command not found Git no instal·lat Linux: sudo apt install git. macOS: brew install git. Windows: Descarregueu des de https://git-scm.com.
Email no verificat a GitHub No heu clicat l'enllaç Reviseu la safata d'entrada i spam. Reenviï l'email des de GitHub si cal.
git push demana credencials contínuament HTTPS sense cache Configureu cache: git config --global credential.helper cache (Linux/macOS) o git config --global credential.helper wincred (Windows).
Error remote: Permission denied Credencials incorrectes Creeu token PAT a GitHub (Settings → Developer settings → Personal access tokens) i utilitzeu-lo com a contrasenya.

6.10. Troubleshooting

Hot reload no funciona Watch mode no actiu Atureu servidor (Ctrl+C) i proveu: ng serve --poll=2000.
Canvis HTML no es reflecteixen Cache navegador Premeu Ctrl+Shift+R (hard refresh) al navegador.
console.log no apareix Consola filtrada A DevTools, assegureu-vos que el filtre és "All levels" (no només Errors).

7.12. Troubleshooting

Problema Causa Solució
git push falla amb error 403 Credencials incorrectes Creeu token PAT a GitHub (Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token). Àmbit: repo. Copieu el token i useu-lo com a contrasenya.
git remote add diu "remote origin already exists" Remote ja configurat Elimineu amb git remote remove origin i torneu a afegir-lo.
Fitxers molt grans node_modules/ sense ignorar Verifiqueu que .gitignore conté /node_modules/. Si va pujar, elimineu-lo del repositori: git rm -r --cached node_modules/ i torneu a fer commit.
Error "updates were rejected" Branch local i remot divergeixen Primera vegada: git push -u origin main --force (NOMÉS si el repositori remot està buit).

Problemes transversals

Símptoma On passa Causa Solució

Unitat 1

Preparació de l'entorn de desenvolupament (RA1)

Descripció de la Unitat

En aquesta unitat aprendreu a preparar l'entorn de desenvolupament per crear aplicacions amb Angular. Identificareu els requisits tècnics, instal·lareu les eines necessàries (Node.js, npm, Angular CLI, VS Code, Git) i creareu el projecte base que utilitzareu durant tot el curs.

En finalitzar, tindreu un projecte Angular funcional amb servidor de desenvolupament actiu, control de versions configurat i l'estructura necessària per començar a desenvolupar components.

1.1

Conceptes fonamentals i instal·lació

Què aprendreu:

  • Descobrir què són els frameworks de costat client i per què s'utilitzen en el desenvolupament web modern
  • Comparar les alternatives més populars del mercat: Angular, React i Vue.js
  • Identificar els requisits tècnics necessaris per desenvolupar aplicacions Angular
  • Instal·lar i configurar Node.js i npm al vostre sistema
  • Verificar la instal·lació i comprendre el rol de l'ecosistema Node.js en Angular
Començar bloc →
1.2

Entorn de desenvolupament integrat

Què aprendreu:

  • Configurar Visual Studio Code amb les extensions essencials per a Angular
  • Aprofitar les funcionalitats de l'IDE: IntelliSense, debugging i terminal integrat
  • Comprendre els fonaments de Git per al control de versions del vostre codi
  • Gestionar repositoris locals amb commits, branques i historial de canvis
  • Utilitzar GitHub com a repositori remot per col·laborar i compartir projectes
Començar bloc →
1.3

Creació del projecte base

Què aprendreu:

  • Instal·lar i configurar Angular CLI per generar i gestionar projectes
  • Crear el vostre primer projecte Angular amb l'estructura de carpetes estàndard
  • Comprendre l'organització de fitxers: components, mòduls, serveis i assets
  • Arrencar el servidor de desenvolupament i explorar la funcionalitat de hot reload
  • Utilitzar comandes CLI bàsiques per crear components, serveis i altres elements

1.4

Cas d'ús: Preparació de l'entorn

Què aprendreu:

  • Aplicar tots els coneixements de la unitat en un projecte integrador complet
  • Crear l'aplicació "Catàleg Elements" des de zero amb Angular CLI
  • Configurar correctament l'entorn de desenvolupament i control de versions amb Git
  • Organitzar l'estructura del projecte seguint les bones pràctiques d'Angular
  • Desplegar l'aplicació al servidor de desenvolupament i verificar el seu funcionament
  • Documentar el procés i preparar el projecte per a les següents unitats

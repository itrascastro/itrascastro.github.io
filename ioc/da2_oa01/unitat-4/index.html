Unitat 4

Navegació i arquitectura d'aplicacions (RA4)

Descripció de la Unitat

En aquesta unitat aprendreu a implementar navegació entre vistes amb el sistema de rutes d'Angular, organitzar aplicacions amb arquitectura escalable i aplicar tècniques d'optimització de rendiment per crear aplicacions professionals llestes per a producció.

Dominareu la configuració de rutes amb paràmetres, guàrdies de navegació, lazy loading de mòduls, gestió d'estat centralitzada, separació de responsabilitats en capes, estratègies de detecció de canvis i desplegament optimitzat per a entorns de producció.

4.1

Sistema de rutes i navegació

Què aprendreu:

  • Implementar el sistema de rutes d'Angular per crear aplicacions SPA
  • Configurar rutes amb paràmetres per crear vistes dinàmiques
  • Utilitzar navegació programàtica per controlar el flux de l'aplicació
  • Crear rutes niades per organitzar jerarquies de vistes
  • Implementar guàrdies de navegació per protegir vistes i validar accessos
Començar bloc →
4.2

Arquitectura escalable

Què aprendreu:

  • Aplicar patrons d'arquitectura escalable amb gestió d'estat centralitzada
  • Organitzar l'aplicació en mòduls amb lazy loading per optimitzar la càrrega
  • Separar responsabilitats en capes: presentació, lògica de negoci i dades
  • Implementar patrons de disseny per crear aplicacions mantenibles
  • Aplicar principis SOLID per a codi net i escalable
Començar bloc →
4.3

Optimització de rendiment

Què aprendreu:

  • Optimitzar el rendiment amb estratègies de detecció de canvis (OnPush)
  • Implementar virtualització de llistes per gestionar grans volums de dades
  • Configurar builds de producció amb compressió i tree-shaking
  • Aplicar tècniques de lazy loading i preloading de mòduls
  • Desplegar l'aplicació en servidors web i plataformes cloud
Començar bloc →
4.4

Cas d'ús: Aplicació completa

Què aprendreu:

  • Integrar tots els conceptes en un cas d'ús pràctic complet
  • Implementar navegació entre múltiples vistes amb sistema de rutes avançat
  • Aplicar arquitectura en capes amb separació de responsabilitats
  • Configurar lazy loading de mòduls per optimitzar la càrrega inicial
  • Aplicar optimitzacions de rendiment i preparar l'aplicació per a producció
